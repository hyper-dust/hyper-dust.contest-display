# Hackathon Interno - Presentazione

Presentazione web realizzata con **Reveal.js** per l'hackathon interno "Lost in the call rooms".

## 🚀 Come usare

### Opzione 1: Apri direttamente il file
Basta aprire `index.html` nel browser.

### Opzione 2: Server locale (consigliato)
```bash
# Con Python
python -m http.server 8000

# Con Node.js
npx serve

# Con PHP
php -S localhost:8000
```

Poi apri `http://localhost:8000` nel browser.

## ⌨️ Controlli tastiera

- **Frecce** → Naviga tra le slide
- **Spazio** → Prossima slide
- **ESC** o **O** → Panoramica di tutte le slide
- **F** → Fullscreen
- **S** → Note del presentatore (se abilitate)
- **B** o **.** → Schermo nero (pausa)

## 🎯 Personalizzazione

Per modificare i colori, modifica le variabili CSS in `style.css`:

```css
:root {
    --primary-color: #00d9ff;
    --secondary-color: #ff006e;
    --accent-color: #ffbe0b;
}
```
